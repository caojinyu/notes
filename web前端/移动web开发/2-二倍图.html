物理像素和物理像素比 物理像素是屏幕显示的最小颗粒，是物理真实存在的，这是厂商在出厂时就设置好了比如苹果6是750 * 1334. 我们开发时候的1px不一定就等于1个物理像素的。 在PC页面，1个px等于一个物理像素，但是移动端就不尽相同。 一个px能显示的物理像素点的个数，称为物理像素比或屏幕像素比。 对于图片，在手机上由于像素比的问题就会被放大模糊。 我们采取二倍图。 比如我们要放一个50 * 50的图片， 我们放一个100 * 100 的图片，然后手动的缩小到50 * 50，那么在手机上渲染的时候，会被放大到100 * 100。 背景图的缩放： background-size： 宽度 高度; // 规定背景图像的尺寸 background-size：500px; // 等比例缩放 background-size: 50%; // 相对于父盒子的百分之。 background-size: cover; //把背景图像对角线拉伸扩展到足够大，完全覆盖完全 background-size: contain; //把图像对角线拉伸扩展到最大，使其宽度或者高度其中之一达到覆盖就不再进行拉伸了。
